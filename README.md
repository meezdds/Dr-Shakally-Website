# Dr. Shakally Website

Personal authority website for Dr. Monzer Shakally, focused on speaking, dental business-model innovation, higher standards of care, applied AI, and digital dentistry workflows.

## Deploy with GitHub and Vercel

1. Import this repository as a new project in Vercel.
2. Keep Vercel's detected Vite settings.
3. Deploy the project.
4. Connect DrShakally.com only after reviewing the preview.

## Local development

```bash
npm install
npm run dev
```

Primary editable website copy is stored in `src/content.js`.

The hero portrait is a generated placeholder and should be replaced with an approved photograph of Dr. Shakally before launch.
